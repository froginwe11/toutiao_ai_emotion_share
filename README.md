<h4><a name="t1"></a><strong>二、操作流程</strong></h4> 
<p><strong>1、素材</strong></p> 
<p>出轨赛道素材搜集，AI写作的爆文方式：我们是“以爆制爆”的二爆玩法。通过这个玩法去搜集素材，也就是搜集已经爆过的对标微头条，然后在AI去续写开头。</p> 
<p>根据指标去选素材，找近期/往期（这个可以测时间段）50W+以上展现的爆文，找千赞文的微头条，优先找点赞数量1000+以上(300/500点赞)，因为我们看不到阅读量，所以需要看点赞量，点赞量越高越好爆，比如下图这篇就是爆款的指标。</p> 
<p><img alt="" height="270" src="https://i-blog.csdnimg.cn/direct/1be67f5207fe4b8bb6a4c31a1adccd5c.png" width="635"></p> 
<p><strong>2、关键词</strong></p> 
<p>要学会搜集相关关键词，初始化词根涵盖：出轨、小三、离婚、家暴....</p> 
<p>通过初始词根，可以拓展更多相关关键词，例如：有染，(婚内)出轨， (抓) 小三，婚外恋，婚外情，外遇，维护小三......</p> 
<p>需要自己去拓展，我只给简单的思路，我没做过小三，也没找过小三，也没抓过小三，很多东西不是很清楚，建议大家一定要养成搜集相关关键词的习惯，这样好快速找对标内容。</p> 
<p><strong>3、找素材</strong></p>

<p><strong>4、养号</strong></p> 
<p>发布之前需要养号，养2-3天，刷下赛道关键词，看看对标，顺便搜集素材，在这里搜，找到内容后，在手机端APP看完对标内容，点赞、评论、收藏就行。</p>

<p><strong>5、AI写作</strong></p> 
<p>超级简单，找素材的时候说了“以爆制爆”的二爆玩法思维，我们是找爆文的开头，然后用AI续写，所以要不断搜集爆文的开头部分，整个操作过程就4个步骤。</p> 
<p><strong>第一步、找爆款</strong>，复制开头，找爆款的4个指标：展现量、点赞量、评论量、阅读量，这个按照上面找素材的方法里讲的去做，复制开头，看案例：</p>
